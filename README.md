### Frontend
#### Setup
```sh
cd frontend
npm install
```

#### Build
```sh
# In `frontend` directory
npm run build
```

### Backend
#### Setup
```sh
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

#### Run
```sh
python3 main.py
```
